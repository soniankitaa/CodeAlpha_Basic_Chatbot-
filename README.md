# CodeAlpha_BasicChatbot

## Project Overview
Basic Chatbot is a rule-based chatbot developed using Python. It interacts with users through predefined responses and demonstrates fundamental concepts of Python programming.

## Features
- User greetings
- Date and time information
- User name recognition
- Calculator functionality
- Chat history tracking
- Help menu
- Session summary
- Input validation

## Technologies Used
- Python 3

## How to Run

1. Clone or download the repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python chatbot.py
```
